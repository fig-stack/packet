# STIFF
Updated version of sniffex (https://www.tcpdump.org/pcap.html).

# Supported Protocols
```
TCP
UDP
ICMP
IP
```

# How to use
```bash
fig@machine:~$ make
fig@machine:~$ sudo ./stiff <number_of_packets>

```
